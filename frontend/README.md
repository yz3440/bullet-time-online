# Bullet Time Online

This is an effort to reconstruct the famous bullet time scene from the movie The Matrix (1999) using gaussian platting, and sharing the scene online.

The source images are extracted from a UHD BluRay copy of the movie. The scene is from 01:45:31 to 01:45:41, totalling 252 frames.

## Process

I reverse engineered the camera path and the camera parameters using the source images and Agisoft Metashape.
